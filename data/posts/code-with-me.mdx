# ¿Cómo usar “Code with me” de IntelliJ?

Esta funcionalidad es muy útil para hacer pair programming, ya existía algo
parecido llamado Live Share para VsCode pero esté viene con Jisti por debajo lo
que proporciona un medio de comunicación propio, explicaremos primero cómo
obtenerlo. Una forma es actualizar a la última versión o descargar el plugin
para la versión del IDE que usted esté usando.

## Crear reunión

Una vez obtenido sale esta opción arriba derecha o en la pestaña de `Tools`:

<MdxImage
  src="/assets/blog/code-with-me/code-with-me-1.png"
  alt="code-with-me-1"
></MdxImage>

Para empezar crearemos nosotros una reunión, para este caso me interesa que la
otra persona pueda editar el código.

<MdxImage
  src="/assets/blog/code-with-me/code-with-me-2.png"
  alt="code-with-me-2"
></MdxImage>

Una vez creado se copiará el link para compartir. Las persona que acceden a
través del link deberán se aceptadas por el anfitrión.

<MdxImage
  src="/assets/blog/code-with-me/code-with-me-3.png"
  alt="code-with-me-3"
></MdxImage>

Una vez aceptada la petición del firewall para dar acceso al participante, este
ingresa a la sala.

<MdxImage
  src="/assets/blog/code-with-me/code-with-me-4.png"
  alt="code-with-me-4"
></MdxImage>

Los participantes pueden escribir con esta opción de editar seleccionada
anteriormente dentro del código.

<MdxImage
  src="/assets/blog/code-with-me/code-with-me-5.png"
  alt="code-with-me-5"
></MdxImage>
Si lo qué necesitas es seguir la pantalla de alguien bastará con hacer clic en
su icono de Jitsi. La pantalla en la qué este el participante se cambiará de
color. Para dejar de seguir volver a dar en el icono.

<MdxImage
  src="/assets/blog/code-with-me/code-with-me-6.png"
  alt="code-with-me-6"
></MdxImage>

## Unirse a la reunión

Podemos unirnos a otras reuniones usando la opción de “Join Another IDE as
Participant…”

<MdxImage
  src="/assets/blog/code-with-me/code-with-me-7.png"
  alt="code-with-me-7"
></MdxImage>

Pegamos el link en la ventana o clicamos en el link y el navegador conectará el
IDE.

<MdxImage
  src="/assets/blog/code-with-me/code-with-me-8.png"
  alt="code-with-me-8"
></MdxImage>
<MdxImage
  src="/assets/blog/code-with-me/code-with-me-9.png"
  alt="code-with-me-9"
></MdxImage>

Una vez el anfitrión nos acepte empezará a descargar los archivos el proyecto.
Una vez acabado ya podrás estar apoyando a tu compañero.

<MdxImage
  src="/assets/blog/code-with-me/code-with-me-10.png"
  alt="code-with-me-10"
></MdxImage>

Ver commits Esta opción aparece cuando el anfitrión de la reunión activa el
versionado con Git.
